About this Project

Project 1 - Travel Blog

This is my contribution for the first Project for the Full-Stack Web Development-Course at the Hamburg Coding School.

It uses no Frameworks, just vanilla HTML, CSS and Javascript.

All Information is stored in the Local Sorage of the Browser, the App fetches information from the APIs:
- Google Maps
- Wikipedia
- Openweathermaps
